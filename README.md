# demka

Markdown → slides. 

1. Open [demka.in.ua](https://demka.in.ua)
2. Write markdown
3. Press **F** to present.
4. Enjoy your demka.

## How it works

- Headings become nodes on a canvas.
- The camera pans and zooms between them as you present. 
- Bullets reveal one at a time.

## Use

- No install.
- No build. 
- Open the URL or clone and open `docs/index.html` in a browser.

## Keys

| Key | Action |
|---|---|
| `F` | Present / exit |
| `Space` `→` | Next bullet, then next node |
| `←` | Previous node |
| `?` | Keyboard help |
